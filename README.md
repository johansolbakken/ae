## Hello World

- Need flex and bison

Make compiler

```bash
clone the repo
cd ae
make
```

example/fib.ae

```c++

int main() {
    int i = 0;
    int a = 0;
    int b = 1;
    println(a);
    println(b);
    while (i < 10) {
        int c = a + b;
        println(c);
        a = b;
        b = c;
        i = i + 1;
    }
}
```

Build program

```bash
./aec < example/fib.ae -o fib
./fib
```
